Welcome to incawrapper's documentation!
=======================================

incawrapper is a Python wrapper for the matlab application INCA. It is a tool for
13C metabolic flux analysis (13C-MFA). The incawrapper package allows to import data,
setup the model and run INCA all from within Python. The results can be exported
back to Python for further analysis and simply saved as .csv files.
